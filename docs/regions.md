# PIA regions

Every value `PIA_LOC` (compose mode) / `LOC` (single-container mode) accepts,
from PIA's live server list. **Port fwd** is whether the region supports
port forwarding (not used by the exit node, listed for completeness).
**Geo** means the server is geo-located: physically hosted elsewhere with an
IP registered to the named region.

Snapshot generated 2026-07-20 (165 regions). PIA adds and removes
regions occasionally; print the current list any time with:

```bash
./scripts/list-regions.sh
```
